## How to Contribute to Changelog

When adding new tools or features, please update this changelog with:

1. **Version number** (following semver)
2. **Date** of release
3. **Category** (Added/Changed/Deprecated/Removed/Fixed/Security)
4. **Description** of changes
5. **Statistics** if applicable

Example:

